# ElvUI
ElvUI build for WoW 3.3.5a. The zip bundles both required components together.

## Download

**[Download ElvUI.zip](https://github.com/GetParanoid/335_Addons/releases/download/latest/ElvUI.zip)** - includes ElvUI and ElvUI_OptionsUI

## What's different in this build

### Spell rank on cast bar
The cast bar displays the rank of the spell being cast (e.g. `Fireball (Rank 11)`). Vanilla ElvUI omits rank information entirely.

## Contents

| Folder | Purpose |
|---|---|
| `ElvUI/` | Core UI framework |
| `ElvUI_OptionsUI/` | In-game configuration interface |

Both folders are required. ElvUI_OptionsUI will not function without ElvUI loaded.

## Installation

Extract the zip into `Interface/AddOns/`. Both `ElvUI/` and `ElvUI_OptionsUI/` are inside - drop them both in.

Type `/ec` in-game to open the ElvUI configuration panel.
